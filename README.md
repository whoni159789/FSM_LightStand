# FSM_LightStand
Vivado 2020.1에서 Verilog을 통한 PWM 구현을 통한 LightStand 코드

## [FSM_LightStnad 동작 원리]
1) Up 버튼 : Light 밝기 증가  
2) Down 버튼 : Light 밝기 감소  
3) Center 버튼 : Light Off
